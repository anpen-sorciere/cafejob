# テストデータ作成ガイド

## テストデータの作成方法

以下のコマンドを実行してテストデータを作成します：

```bash
cd C:\xampp\htdocs\cafejob-laravel
php artisan db:seed --class=TestDataSeeder
```

または、すべてのシーダーを実行する場合：

```bash
php artisan db:seed
```

## 動作確認

テストデータ作成後、以下を確認してください：

1. **ホームページ** (`http://localhost:8000`)
   - 統計データが表示される
   - 人気ランキングが表示される

2. **求人一覧ページ** (`http://localhost:8000/jobs`)
   - 5件の求人が表示される
   - 検索・フィルターが動作する

3. **求人詳細ページ** (`http://localhost:8000/jobs/{id}`)
   - 求人詳細が表示される
   - キープ機能が動作する
   - 応募機能が動作する

4. **キープ機能**
   - ログイン後、キープボタンをクリック
   - トースト通知が表示される
   - ボタンの状態が更新される

## 注意事項

- 既に同じ名前の店舗・求人が存在する場合は、新規作成されません（`firstOrCreate`を使用）
- 都道府県データが存在しない場合はエラーになります。先に`PrefectureSeeder`を実行してください
